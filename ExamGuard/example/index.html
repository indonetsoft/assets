Media Control — Simulasi Tombol Next

⏭️ Simulasi Navigasi Soal

Putar salah satu media di atas, lalu tekan salah satu tombol berikut untuk mensimulasikan user menekan Next sebelum media selesai.

Penjelasan masing-masing:
⏸ lastPause() — Pause media. Posisi tetap, bisa di-resume jika user kembali ke soal ini.
⏹ lastStop() — Stop + reset ke awal. Counter tidak berubah.
⏹+1 lastCounter() — Stop + reset ke awal + tambah counter +1. Cocok jika meninggalkan media dihitung sebagai 1x play.
▶ playFirstMedia() — Play media pertama yang visible di viewport dan belum mencapai maxPlay.
mediaControls & mediaPlayButton

Ketika mediaControls: false, ExamGuard menyembunyikan kontrol bar native browser dan secara otomatis menyisipkan baris kontrol di bawah setiap elemen <audio> dan <video> — tanpa perlu markup tambahan di HTML. Cocok untuk konten yang dimasukkan via WYSIWYG editor (CKEditor, TinyMCE, dsb.).

mediaControls: true

Kontrol bar native browser tampil penuh. ExamGuard tidak mengubah tampilan — hanya membatasi jumlah pemutaran di belakang layar. (Default)

Catatan teknis:
• Baris kontrol (.counter-row) dibuat otomatis oleh ExamGuard — tidak perlu markup di HTML.
• Progress bar menggunakan <progress value max> native dengan satuan detik — tidak ada animasi CSS yang menyebabkan efek "jalan-jalan".
• Counter (n/max) disinkronkan langsung dari storage saat event play, pause, dan ended.
• CSS seluruh komponen di-inject otomatis via <style id="exam-guard-styles"> — tidak perlu file CSS terpisah.
Cara Penggunaan

1. Browser — via <script> tag

Cara paling sederhana. Cocok untuk sistem CBT tanpa build tool.

<!-- Gunakan versi minified untuk production -->
<script src="dist/exam-guard.min.js"></script>

<div class="exam-media">
  <audio src="listening1.mp3" controls></audio>
  <video src="video1.mp4" controls></video>
  <iframe src="https://www.youtube.com/embed/VIDEO_ID?enablejsapi=1"></iframe>
</div>

<script>
  new ExamGuard({
    selector           : ".exam-media",
    sessionToken       : "TOKEN_DARI_SERVER",  // wajib, unik per peserta
    maxPlay            : 2,
    fullscreenMode     : true,
    disableCopy        : true,
    disableTabSwitch   : true,
    preventMultipleTab : true,
    forcePlayUntilEnd  : false,
    lockOnViolation    : false,
    countOnPause       : false,
    saveLastSeek       : false,  // restore posisi seek setelah refresh
    ytClean            : true,
    ytOverlay          : false,
    devMode            : false,  // jangan aktifkan di production

    onViolation(type, message) {
      console.warn("Pelanggaran:", type, message);
    },

    onMediaBlocked(mediaKey) {
      console.warn("Media diblokir:", mediaKey);
    },
  });
</script>

2. NPM / ES Module

Untuk proyek dengan bundler seperti Vite, Webpack, atau Rollup.

// Install
npm install exam-guard

// Penggunaan
import ExamGuard from "exam-guard";

const guard = new ExamGuard({
  selector           : ".exam-media",
  sessionToken       : "TOKEN_DARI_SERVER",
  maxPlay            : 2,
  fullscreenMode     : true,
  disableCopy        : true,
  disableTabSwitch   : true,
  preventMultipleTab : true,
  forcePlayUntilEnd  : false,
  lockOnViolation    : false,
  countOnPause       : false,
  saveLastSeek       : false,
  ytClean            : true,
  ytOverlay          : false,
  devMode            : false,

  // Callback kustom (opsional)
  onViolation(type, msg) {
    console.warn("Pelanggaran:", type, msg);
  },
  onMediaBlocked(key) {
    console.warn("Media diblokir:", key);
  },
});

3. Framework7 Plugin

Integrasi otomatis dengan siklus hidup halaman Framework7.

import ExamGuard         from "./src/exam-guard-core.js";
import ExamGuardF7Plugin from "./src/exam-guard-f7-plugin.js";
import Framework7        from "framework7";

const app = new Framework7({
  // Daftarkan plugin
  plugins: [ExamGuardF7Plugin],

  // Konfigurasi ExamGuard
  examGuard: {
    selector           : ".exam-media",
    sessionToken       : "TOKEN_DARI_SERVER",
    maxPlay            : 2,
    fullscreenMode     : true,
    disableCopy        : true,
    disableTabSwitch   : true,
    preventMultipleTab : true,
    forcePlayUntilEnd  : false,
    lockOnViolation    : false,
    countOnPause       : false,
    saveLastSeek       : false,
    ytClean            : true,
    ytOverlay          : false,
    devMode            : false,  // DEV ONLY — aktifkan shortcut Ctrl+Shift+A+E

    onViolation(type, message) {
      console.warn("Pelanggaran:", type, message);
    },

    onMediaBlocked(mediaKey) {
      console.warn("Media diblokir:", mediaKey);
    },
  }
});

// ExamGuard akan otomatis aktif di halaman yang mengandung .exam-media
// dan dibersihkan saat halaman dihapus dari DOM.
Referensi Konfigurasi
Parameter Default Keterangan
selector .exam-media CSS selector container media
sessionToken required Token sesi unik per peserta dari server. Wajib diisi.
maxPlay 2 Batas jumlah pemutaran per media
fullscreenMode true Paksa fullscreen saat ujian. Keluar fullscreen = pelanggaran.
disableCopy true Blokir copy, cut, klik kanan, Ctrl+C/V/U/S/A, dan F12
disableTabSwitch true Deteksi perpindahan tab via Visibility API
preventMultipleTab true Blokir tab ganda via BroadcastChannel (fallback localStorage)
forcePlayUntilEnd false Media tidak boleh di-pause setelah diputar
lockOnViolation false Hentikan dan kunci ujian jika ada pelanggaran
countOnPause false Tambah counter saat media di-pause. false = counter hanya bertambah saat play dari awal atau setelah ended. true = setiap pause dianggap akhir sesi, resume berikutnya dihitung play baru.
saveLastSeek false Simpan posisi terakhir media ke localStorage dan restore otomatis saat halaman di-refresh. Berlaku untuk audio, video, dan YouTube.

Posisi disimpan tiap timeupdate (audio/video) atau tiap 1 detik via interval (YouTube).
Seek position dihapus otomatis saat media selesai (ended) atau saat clearSession() dipanggil.
ytClean true Bersihkan UI YouTube embed. Menyembunyikan kontrol player, logo, rekomendasi, anotasi, tombol fullscreen, dan keyboard shortcut. Serta memasang overlay untuk mencegah klik langsung ke player (logo, title, share, tonton nanti).
ytOverlay false false — Pasang 2 overlay tipis: atas 25% dan bawah 25%. Area tengah 50% tetap terbuka sehingga tombol play native YouTube masih bisa diklik. Overlay atas memblokir logo & title, overlay bawah memblokir share & rekomendasi.

true — 1 overlay penuh menutup seluruh player. Semua klik ditangani via IFrame API (toggle play/pause). Jika forcePlayUntilEnd: true, klik diabaikan sepenuhnya.
devMode false Mode developer. Aktifkan shortcut Ctrl+Shift+A+E untuk reset semua counter media sesi. Jangan aktifkan di production.
onViolation null Callback (type, message) saat pelanggaran. Jika null, tampil alert.
onMediaBlocked null Callback (mediaKey) saat media diblokir. Jika null, tampil alert.
